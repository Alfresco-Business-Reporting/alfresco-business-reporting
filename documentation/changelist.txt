Postgresql9.2 support (probably lower/higher versions too)

Field size -> docsize [Oracle only]
Field type -> object_type (if it had existed anyway)

Booleans map to TINYINT or SMALLINT or NUMBER(1). true=1, false=0

Column names (table names too) will be truncated at 30 (Oracle)/64 (MySQL/Postgres) char. Take this into account when creating Audit framework applications. The table name will become the concat of all keys (Application key, AuditPath key, AuditPath key etc.)
